1. Include Skill Area + Concept
Format: SkillArea_Concept_Practice.md

Example:

Python_DSA_LinkedList_Practice.md

Docker_Networking_Practice.md

React_Hooks_Practice.md

2. Add Date or Iteration (Optional)
Helps track progress over time.

Format: SkillArea_Concept_YYYY-MM-DD.md or SkillArea_Concept_v1.md

Example:

Python_DSA_Recursion_2026-01-03.md

Kubernetes_Pods_v2.md

3. Separate Hands-On vs Skill Upgrade
If you want to keep the earlier structure intact:

HandsOn_Python_DSA_Recursion.md

SkillUpgrade_Docker_CourseNotes.md

4. Keep It Short but Descriptive
Avoid long names like Python_DataStructuresAndAlgorithms_RecursionPracticeWithExamples.md.

Instead: Python_DSA_Recursion.md.
